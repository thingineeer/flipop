## Phase별 결정

### [1-1] 초반 2색 모드 도입
- 날짜: 2026-03-15
- Phase: 1
- 상태: 결정됨
- 맥락: 3색 초반이 너무 어려워 첫 줄 클리어까지 10턴+ 소요. 유저 직접 피드백 "넘 어려워"
- 결정: score 0~299에서 Red, Blue 2색만 사용. 300점 이상에서 3색 전환
- 대안: 3색 유지 + 초기 그리드에 거의 완성된 줄만 배치
- 결과: 2색이면 탭 1회로 줄 클리어 확률 대폭 상승. 초반 성취감 보장

### [1-2] 가로 클리어 5칸→3+ 연속으로 변경
- 날짜: 2026-03-15
- Phase: 1
- 상태: 결정됨
- 맥락: 5칸 전부 같은 색 맞추기는 거의 불가능. 세로(3+)와 불균형
- 결정: 가로도 세로와 동일하게 연속 3개 이상 같은 색이면 클리어
- 대안: 4칸 매칭, 또는 T자/L자 형태 매칭
- 결과: 클리어 빈도 대폭 증가. 게임 체감 난이도 하락. 중앙 빈 공간 문제 완화

### [1-3] 타이머 90초→120초 + 콤보 보너스 상향
- 날짜: 2026-03-15
- Phase: 1
- 상태: 결정됨
- 맥락: 90초는 초보자에게 시간 압박이 심함
- 결정: 120초 + 콤보 보너스 +줄당 2초 추가
- 대안: 타이머 제거 (무한 모드)
- 결과: 초보자 여유로운 학습, 숙련자는 콤보로 무한 연장 가능

### [1-4] 난이도 곡선 4단계 (Phase A~D)

- Phase: 1
- 상태: 결정됨
- 맥락: 기존 2단계 난이도(500점, 1000점)로는 점진적 도전감 부족
- 결정: 0~299(2색/5턴) → 300~699(3색/4턴) → 700~1499(3색/3턴) → 1500+(4색/2턴)

- 결과: 자연스러운 난이도 곡선, "한판 더" 충동 증가
