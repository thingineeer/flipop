# Phase 2: 온보딩 완전 재설계

> **목표**: 텍스트 읽기 0 → 플레이하면서 자연스럽게 메커닉 학습
> **핵심 파일**: `lib/ui/onboarding_overlay.dart`, `lib/ui/game_screen.dart`, `lib/ui/welcome_screen.dart`
> **의존**: Phase 1 (게임 밸런스) 완료 후

---

## 현재 문제

1. **WelcomeScreen**: 3페이지 캐러셀이지만 "읽어야 하는" 텍스트 위주 → 대부분 스킵
2. **OnboardingOverlay**: 3단계 (탭→클리어→콤보) 정적 이미지 → 실제 플레이와 괴리
3. **첫 게임 진입 후**: 아무 가이드 없이 5×6 랜덤 그리드 → 당황
4. **"탭하면 주변 색이 바뀐다"**는 한 줄 설명만으로는 메커닉 이해 불가

---

## 새로운 온보딩 설계: "가이디드 퍼즐" 방식

### Step 1: 미니 퍼즐 (첫 실행 시)

WelcomeScreen 대신 **3개의 미니 퍼즐**로 메커닉 학습:

```
퍼즐 1 — "탭의 효과" (3×3 그리드)
┌─────────────────────────────┐
│  🐶  🐱  🐶                │
│  🐱  ✨  🐱    ← "가운데를 탭해보세요!"  │
│  🐶  🐱  🐶                │
│                              │
│  손가락 애니메이션이 가운데를 가리킴    │
│  탭 → 상하좌우 색 변환 확인          │
│  "잘했어요! 탭하면 주변 블록이 바뀌어요" │
└─────────────────────────────┘

퍼즐 2 — "줄 맞추기" (5×2 그리드)
┌─────────────────────────────┐
│  🐱  🐱  🐶  🐱  🐱        │
│  🐶  🐶  🐱  🐶  🐶        │
│                              │
│  화살표가 🐶 위치를 가리킴           │
│  "가로줄을 같은 색으로 맞춰보세요!"   │
│  탭 → 줄 클리어 → POP 이펙트!       │
│  "완벽해요! 가로줄을 맞추면 터져요!"  │
└─────────────────────────────┘

퍼즐 3 — "콤보" (5×3 그리드)
┌─────────────────────────────┐
│  두 줄이 거의 완성된 상태           │
│  한 번 탭 → 연쇄 클리어!           │
│  "대단해요! 연쇄로 터지면 콤보!"     │
│  COMBO x2 이펙트 표시              │
│  "콤보가 높을수록 보너스 시간!"      │
└─────────────────────────────┘
```

### Step 2: 가이디드 첫 게임

미니 퍼즐 완료 후, 실제 게임 진입하되:

```
처음 3턴:
  - "거의 완성된 줄" 하이라이트 (Phase 1의 힌트 시스템 활용)
  - 말풍선: "반짝이는 줄을 탭해서 완성해보세요!"
  - 타이머 일시정지 상태

3턴 이후:
  - "준비됐나요? 게임 시작!" 카운트다운
  - 타이머 시작
  - 힌트는 유지하되 말풍선 제거
```

### Step 3: 점진적 기능 해금

```
첫 게임:     기본 메커닉만 (탭 + 클리어)
2번째 게임:  콤보 시스템 소개 팝업 (1줄)
3번째 게임:  리워드 광고 기능 해금 안내
5번째 게임:  리더보드 참여 안내
```

## 구현 상세

### 새 파일: `lib/ui/tutorial_screen.dart`

```dart
// 미니 퍼즐 3개를 순차 진행하는 전용 화면
// - 축소된 그리드 (3x3, 5x2, 5x3)
// - 가이드 오버레이 (손가락 애니메이션, 말풍선)
// - GameState를 그대로 활용하되 타이머/점수 비활성화
// - 완료 시 SecureStorage에 기록
```

### 수정: `lib/ui/onboarding_overlay.dart`

```
기존 3단계 정적 오버레이 → 제거 또는 "가이디드 첫 게임" 오버레이로 교체
- Step별 말풍선 + 하이라이트
- 타이머 일시정지 로직
- 3턴 후 자동 해제
```

### 수정: `lib/ui/welcome_screen.dart`

```
기존 3페이지 캐러셀 → 1페이지 간소화
- 캐릭터 인사 + "바로 시작!" 버튼
- 소셜 로그인은 게임 후 MoreScreen에서 유도
- 첫 진입: WelcomeScreen → TutorialScreen → GameScreen
```

### 수정: `lib/ui/game_screen.dart`

```
첫 게임 감지 (totalGamesPlayed == 0):
  - 처음 3턴: 가이디드 모드
  - 힌트 하이라이트 활성화
  - 타이머 일시정지
```

### 수정: `lib/main.dart` (AuthGate 라우팅)

```
기존: Welcome → Home → Nickname → Game
변경: Welcome(간소) → Tutorial(미니퍼즐) → Game(가이디드) → 이후 일반
```

## 구현 순서

```
1. tutorial_screen.dart — 미니 퍼즐 화면 신규 생성
2. welcome_screen.dart — 1페이지 간소화
3. main.dart — 라우팅 수정 (Tutorial 추가)
4. onboarding_overlay.dart — 가이디드 첫 게임 오버레이로 변환
5. game_screen.dart — 가이디드 모드 (첫 3턴 타이머 정지 + 힌트)
6. l10n/app_*.arb — 4개 언어 텍스트 추가
7. flutter gen-l10n
8. flutter analyze + flutter test
```

## 성공 기준

- 미니 퍼즐 3개 완료 시간: 30초~1분
- 첫 게임에서 최소 1줄 클리어 성공률: 95%+
- 온보딩 이탈률: 10% 이하 (기존 추정 40%+ → 대폭 감소)
- 텍스트 읽기 없이도 게임 메커닉 100% 이해
