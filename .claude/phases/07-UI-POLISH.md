## A. 블록 애니메이션 강화

### A-1. 탭 피드백
```
현재: 스케일 애니메이션만
변경:
  - 탭한 블록: 살짝 눌림 (scale 0.9 → 1.0, 100ms)
  - 영향받는 인접 블록: 색 변환 시 flip 애니메이션 (Y축 회전 180°, 200ms)
  - 변환 완료: 살짝 바운스 (overshoot curve)
```

### A-2. 클리어 이펙트
```
현재: PopParticle (6개 파티클)
변경:
  - 파티클 12개로 증가 + 크기 다양화
  - 블록 사라질 때: 팽창 → 반투명 → 사라짐 (200ms)
  - 클리어 라인 따라 수평 광선 이펙트 (흰색, 150ms)
  - 콤보 x3+: 화면 전체 짧은 플래시 (흰색 오버레이, 100ms)
```

### A-3. 중력 애니메이션
```
현재: 즉시 이동 (애니메이션 없음으로 추정)
변경:
  - 블록이 떨어질 때 바운스 곡선 (300ms)
  - 착지 시 미세한 squash-and-stretch
  - 여러 블록 동시 낙하: 0.05초 딜레이로 캐스케이드
```

## 구현 순서

```
1. lib/ui/block_widget.dart — 탭 피드백 + 플립 애니메이션
2. lib/ui/pop_particle.dart — 파티클 12개 + 광선 이펙트
3. lib/ui/game_screen.dart — 중력 바운스, 콤보 플래시
4. lib/ui/game_over_overlay.dart — 블러 배경 + 카운팅 + NEW BEST
5. lib/ui/game_screen.dart — 타이머 그라데이션 + 점수 카운팅
6. lib/game/game_colors.dart — 다크 모드 팔레트
7. lib/ui/more_screen.dart — 다크 모드 토글
8. pubspec.yaml — google_fonts (선택)
9. l10n — "다크 모드" 텍스트 4개 언어
10. flutter gen-l10n
11. flutter analyze + flutter test
12. .claude/context/ 업데이트
```

## 성능 주의

- AnimationController는 반드시 dispose()
- RepaintBoundary로 애니메이션 영역 격리
- 파티클 개수 12개 이하 유지 (저사양 기기)
- 다크 모드 전환 시 setState 최소화 (Theme.of 활용)
