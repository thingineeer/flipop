# Phase 1: 게임 밸런스 재설계

> **목표**: "30초만에 재미 → 3분간 몰입 → 다시 한판 충동" 루프 완성
> **핵심 파일**: `lib/game/game_state.dart`
> **제약**: GameState 불변성 유지, 기존 테스트 깨지지 않게

---

## 현재 문제 분석

### 스크린샷 기반 진단:
1. **빈 그리드 + 양쪽 끝에만 블록** → 탭해도 영향 범위 좁음, 전략 불가
2. **3색 초반에도 가로줄 맞추기 매우 어려움** → 5칸 모두 같은 색 확률 극히 낮음
3. **새 줄 추가 빈도 높음** (3턴마다) → 클리어 못하면 순식간에 오버플로
4. **타이머 90초 + 보너스 적음** → 초보자는 시간 압박에 패닉
5. **콤보 외 보상 없음** → 작은 성취감 부재

### 핵심 수치 (game_state.dart 기준):
```
초기 색상: 3개 (Red, Blue, Yellow)
그리드: 5x7 (6행 가시 + 1행 버퍼)
새 줄 추가: 3턴마다 (score < 500)
클리어 조건: 가로줄 5칸 동일색 (사실상 3+ 연속)
타이머: 90초
콤보 보너스: +3/+5/+7/+10초
```

## 변경 사항

### 1. 클리어 조건 완화 — "3-match"에서 시작

**현재**: `_checkAndClearLines()`에서 가로/세로 3+ 연속만 클리어
**변경**:

```
Phase A (score 0~299):    3연속 클리어 OK + 2색만 사용
Phase B (score 300~699):  3연속 클리어 + 3색
Phase C (score 700~1499): 3연속 클리어 + 3색 + 새줄 빈도 증가
Phase D (score 1500+):    3연속 클리어 + 4색 + 빠른 새줄
```

**구현 포인트**:
- `_applyDifficulty()` 메서드에 Phase A 추가
- Phase A에서는 **2색만** (Red, Blue) → 맞추기 매우 쉬움
- 첫 30초간 "성공 경험" 보장

### 2. 초기 그리드 개선 — "거의 완성된 줄" 배치

**현재**: 완전 랜덤 초기 배치
**변경**:

```dart
// 초기 그리드 생성 시:
// - 하단 2줄: 4/5칸 같은 색 (1칸만 다름) → "한 번 탭하면 클리어" 보장
// - 중단 2줄: 3/5칸 같은 색 → 약간의 전략 필요
// - 상단: 빈 공간
```

**구현**: `GameState.initial()` 팩토리에 `_generateBalancedGrid()` 추가

### 3. 새 줄 추가 빈도 조정

**현재**:
```
score < 500:  3턴마다
score < 1000: 2턴마다
score 1000+:  2턴마다 + 4색
```

**변경**:
```
score < 300:  5턴마다 (여유로운 학습)
score 300~699: 4턴마다
score 700~1499: 3턴마다
score 1500~2999: 2턴마다 + 4색
score 3000+: 2턴마다 + 4색 + 특수 블록(잠금블록)
```

### 4. 타이머 밸런스

**현재**: 90초 고정, 콤보 보너스만
**변경**:
```
기본 시간: 120초 (여유롭게 시작)
콤보 보너스: +3/+5/+8/+12초 (상향)
줄 클리어 보너스: +2초 (콤보와 별개)
"위기 모드" (30초 이하): 화면 테두리 붉은 펄스 + BGM 빨라짐(향후)
```

### 5. 점수 체계 개선 — 작은 보상 추가

**현재**: 줄 클리어 시에만 점수
**변경**:
```
줄 클리어:     100 × 줄수 × 콤보배율 (유지)
퍼펙트 클리어: 줄 전체가 한 번에 → 보너스 +200
연쇄 클리어:   중력 후 자동 클리어 시 보너스 +150
터치 효율:     10턴 이내 줄 클리어 → 보너스 +50
"거의 완성":   4/5칸 같은 색 줄 → 시각적 힌트 (점수는 아님)
```

### 6. 시각적 힌트 시스템

**새 기능**: 거의 완성된 줄(4/5칸 동일색) 감지 → 해당 줄 살짝 글로우
- `game_state.dart`에 `List<int> get nearCompleteRows` 게터 추가
- UI에서 해당 행에 미세한 하이라이트 표시
- 초보자가 "어디를 탭해야 하는지" 자연스럽게 유도
