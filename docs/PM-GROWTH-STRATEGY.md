# FLIPOP 성장 전략 문서

> **버전**: v1.0 | **작성일**: 2026-03-12 | **상태**: Draft
>
> Flutter 기반 모바일 캐주얼 퍼즐 게임. 5x7 그리드에서 탭→인접 색 순환→가로줄 클리어→중력→연쇄콤보.
> 4종 픽셀 아트 캐릭터 (고양이/강아지/토끼/개구리). Firebase 익명 인증 + Firestore 순위표.

## 1. 유입 전략 (User Acquisition)

### 1-1. ASO 최적화 전략

**앱 이름 후보**
| 후보 | 이유 |
|------|------|
| FLIPOP - 색깔 퍼즐 | 메커닉(flip) + 효과(pop) 직관적 전달 |
| FLIPOP: Color Chain Puzzle | 영문 서브타이틀로 글로벌 검색 대응 |

**핵심 키워드 (한국)**
- 1차: 퍼즐 게임, 색깔 맞추기, 캐주얼 퍼즐, 블록 게임, 두뇌 게임
- 2차: 중독 퍼즐, 무료 퍼즐, 귀여운 퍼즐 게임, 동물 퍼즐
- 롱테일: 색 바꾸기 게임, 줄 맞추기, 콤보 퍼즐, 픽셀 퍼즐 게임

**핵심 키워드 (글로벌)**
- 1차: color puzzle, block puzzle, casual puzzle, brain game, match puzzle
- 2차: cute puzzle, pixel game, chain combo, addictive puzzle
- 롱테일: color flip game, line clear puzzle, animal block game

**스크린샷 전략 (5장)**
1. **히어로 샷**: 콤보 x3 터지는 순간 + POP 파티클 + "한 번의 탭으로 연쇄 폭발!"
2. **캐릭터 샷**: 4종 픽셀 캐릭터 전체 + "귀여운 친구들과 함께"
3. **메커닉 샷**: 탭 전/후 비교 (온보딩 일러스트 활용) + "탭 한 번, 색이 바뀐다"
4. **콤보 샷**: COMBO x3 UI + 쉐이크 이펙트 표현 + "연쇄! 연쇄! 연쇄!"
5. **순위표 샷**: 랭킹 화면 + "전 세계 플레이어와 경쟁"

**앱 설명 구조**
- 첫 3줄: 핵심 메커닉 + 차별점 (탭→색 변화→줄 완성→콤보 폭발)
- 중간: 특징 불릿 (4종 캐릭터, 글로벌 순위, POP 이펙트, 직관적 조작)
- 하단: 키워드 자연 삽입 ("두뇌 퍼즐을 좋아하신다면...")

### 1-2. 바이럴 루프 설계

**점수 공유 카드 (P0 - 가장 중요)**
```
┌─────────────────────────┐
│  FLIPOP                 │
│  ─────────────────────  │
│  🐱 SCORE: 2,400       │
│  COMBO x5 달성!         │
│  ─────────────────────  │
│  [5x5 미니 그리드 미리보기] │
│  ─────────────────────  │
│  나도 도전하기 → [딥링크]  │
└─────────────────────────┘
```
- 게임 오버 시 "공유하기" 버튼 → 이미지 생성 → SNS 공유
- 미니 그리드 스냅샷으로 시각적 호기심 유발
- 딥링크로 스토어 직행

**초대 시스템 (P1)**
- 초대 코드 → 양쪽 모두 보상 (리워드: 특별 캐릭터 스킨 해금)
- Firebase Dynamic Links 활용
- 초대 수 3/5/10명 달성 시 한정 스킨 제공

**SNS 챌린지 트리거**
- COMBO x5+ 달성 시 "MEGA COMBO!" 팝업 + 자동 공유 제안
- 일간 최고 점수 갱신 시 "NEW RECORD!" 공유 제안
- 특정 점수 달성 마일스톤 (1K, 5K, 10K) → 배지 + 공유

## 2. 리텐션 강화 기능 (Retention Features)

### 2-1. 데일리 챌린지 / 미션 시스템

**데일리 챌린지 (P0)**
- 매일 UTC 00:00에 동일한 시드 기반 보드 생성 → 전 유저 같은 퍼즐
- "오늘의 보드" 탭 추가 (메인 화면 상단)
- 모든 유저가 같은 보드를 풀고 점수 비교 → 데일리 순위표 분리
- **목표**: "매일 1판" 습관 형성

**일간 미션 (P1)**
| 미션 | 보상 |
|------|------|
| 1판 플레이 | 코인 50 |
| 콤보 x2 이상 1회 | 코인 30 |
| 500점 이상 달성 | 코인 80 |
| 줄 5개 클리어 | 코인 50 |
| 게임 공유 1회 | 코인 100 |

**주간 미션 (P2)**
- 7일 연속 플레이, 누적 3,000점, 콤보 x5 달성 등
- 주간 미션 완료 시 한정 캐릭터 스킨 조각

### 2-2. 시즌 / 이벤트 시스템

**시즌 패스 (P2)**
- 4주 1시즌, 무료 트랙 + 프리미엄 트랙
- 레벨 1~30, 데일리 미션 + 일반 플레이로 XP 획득
- 무료: 코인, 기본 스킨 / 프리미엄: 한정 캐릭터, 특별 이펙트, 보드 테마

**시즌 테마 로테이션**
| 시즌 | 테마 | 특별 요소 |
|------|------|----------|
| S1 | 봄 꽃놀이 | 벚꽃 파티클, 핑크 보드 |
| S2 | 여름 바다 | 물방울 POP, 비치 캐릭터 |
| S3 | 할로윈 | 호박 블록, 유령 이펙트 |
| S4 | 크리스마스 | 눈 파티클, 산타 스킨 |

**이벤트 (P2)**
- 주말 더블 코인
- "콤보 페스티벌" (콤보 점수 x2, 48시간 한정)
- 신규 캐릭터 출시 기념 한정 챌린지

### 2-3. 캐릭터 수집 / 커스터마이제이션

**현재 자산**: 고양이(빨강), 강아지(파랑), 토끼(노랑), 개구리(초록) — 4종

**확장 계획 (P1~P2)**

| 티어 | 획득 방법 | 예시 |
|------|----------|------|
| 기본 (4종) | 시작 시 제공 | 현재 캐릭터 |
| 레어 (8종) | 코인 구매 / 미션 보상 | 펭귄, 햄스터, 다람쥐, 부엉이... |
| 에픽 (4종) | 시즌 보상 / 이벤트 한정 | 골드 고양이, 우주비행사 강아지... |
| 레전더리 (2종) | 초대 10명 / 특별 업적 | 무지개 유니콘, 드래곤 |

**커스터마이제이션 요소**
- 캐릭터 스킨 (블록에 표시되는 이미지)
- 보드 테마 (배경색 + 그리드 스타일)
- POP 이펙트 테마 (기본/별/하트/불꽃)
- 콤보 사운드 팩

### 2-4. 소셜 기능

**친구 대전 (P2)**
- 같은 시드 보드를 동시에 풀고 점수 비교
- 비동기 방식: 먼저 푼 사람의 점수를 "고스트"로 표시
- Firebase Realtime Database로 실시간 진행 상황 공유

**공유 카드 (P0 — UA와 연결)**
- 게임 오버 시 자동 생성
- 점수, 콤보, 캐릭터, 미니 보드 포함
- share_plus 패키지로 SNS 직접 공유

**친구 순위표 (P1)**
- 글로벌 순위표와 별도로 친구 순위표
- Firebase Dynamic Links로 친구 추가
- 주간 친구 랭킹 리셋 → 매주 경쟁 유도

## 3. 수익화 전략 (Monetization)

### 3-1. 광고 모델

**보상형 광고 (Rewarded — 핵심 수익원)**
| 트리거 | 보상 | 예상 노출 |
|--------|------|----------|
| 게임 오버 → "이어하기" | 현재 보드에서 row 2줄 제거 후 재개 | 유저당 일 1~2회 |
| 스트릭 보호 | 끊어진 스트릭 1회 복구 | 유저당 일 0~1회 |
| 데일리 보너스 x2 | 코인 보상 2배 | 유저당 일 1회 |
| 힌트 (P2) | "이 블록을 탭하면 줄 완성" 가이드 | 유저당 일 0~2회 |

**인터스티셜 광고 (Interstitial)**
- **타이밍**: 게임 오버 → 순위 제출 후 → "PLAY AGAIN" 누르기 전
- **빈도 제한**: 5분에 1회, 하루 최대 6회
- **절대 금지**: 플레이 중, 콤보 진행 중, 온보딩 중
- **쿨타임**: 보상형 광고 시청 후 3분간 인터스티셜 미노출

**배너 광고: 사용하지 않음**
- 5x7 그리드 + UI 요소로 화면 공간 부족
- 배너가 게임 경험을 해치는 캐주얼 퍼즐 특성상 배제

## 4. 기능 로드맵 (우선순위 P0~P3)

| P0 | **연속 플레이 스트릭** (SharedPreferences + UI) | 하 | D7 리텐션 +5~10% |
| P0 | **데일리 챌린지** (시드 기반 보드 + 데일리 순위) | 중 | 매일 복귀 동기 |

| 우선순위 | 기능 | 구현 난이도 | 기대 효과 |
|---------|------|-----------|----------|

## 5. 노스스타 메트릭

### 5-1. 핵심 지표 정의

> **노스스타 메트릭: 일간 활성 플레이 수 (Daily Active Plays)**
>
> "하루에 완료된 총 게임 수"
>
> 이 지표가 올라가면 리텐션(반복 플레이), 인게이지먼트(세션당 플레이 수),
> 유저 베이스(DAU) 모두가 건강하다는 의미.

### 5-2. 서브 메트릭

**유입 (Acquisition)**
| 메트릭 | 설명 |
|--------|------|
| Daily Installs | 일간 신규 설치 수 |
| Organic Rate | 전체 설치 중 오가닉 비율 |
| CPI | 유료 채널 설치당 비용 |

**활성화 (Activation)**
| 메트릭 | 설명 |
|--------|------|
| Onboarding Completion | 온보딩 3단계 완료율 |
| First Game Completion | 첫 게임 완료율 (≠ 이탈) |
| D0 2nd Play Rate | 설치 당일 2판 이상 플레이 비율 |

**리텐션 (Retention)**
| 메트릭 | 설명 |
|--------|------|
| D1 / D7 / D30 Retention | 클래식 리텐션 |
| Streak Rate | 3일+ 연속 플레이 유저 비율 |
| Weekly Active Days | 주당 평균 활성 일수 |

**수익 (Revenue)**
| 메트릭 | 설명 |
|--------|------|
| ARPDAU | 일간 활성 유저당 매출 |
| Ad Revenue per DAU | 광고 매출 / DAU |
| Subscription Conversion | 무료→구독 전환율 |
| LTV (30d / 90d) | 유저 생애 가치 |

**바이럴 (Referral)**
| 메트릭 | 설명 |
|--------|------|
| Share Rate | 게임 오버 후 공유 비율 |
| Invite Conversion | 초대 링크 클릭→설치 전환율 |
| Viral k-factor | 유저 1명이 데려오는 신규 유저 수 |
